# Документация Ursasstube (актуально на 14 мая 2026)

В папке `docs/` оставлены только рабочие и поддерживаемые документы.

## Актуальные документы

- `debug-guide.md` — практический гайд по диагностике runtime/gameplay/store/network.
- `phaser-full-migration-plan.md` — текущий план миграции/развития Phaser-части.
- `phaser-parity-checklist.md` — чеклист паритета поведения между рендерами.
- `onboarding-frontend-implementation-plan.md` — план и заметки по frontend-onboarding.
- `player-menu-smoke.md` — smoke-проверки Player Menu и связанных UX-потоков.
- `telegram-analytics.md` — события и контракт аналитики для Telegram-канала.

## Что очищено

Из репозитория удалены устаревшие или уже выполненные артефакты:
- разовые отчёты, аудиты, ревью и gate-статусы с датами;
- временные release/runbook-документы под закрытые инциденты;
- snapshot JSON-отчёты, которые не являются источником правды для текущей разработки.

Если понадобится исторический контекст, его лучше смотреть в Git history по удалённым файлам.
