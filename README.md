# bageus.github.io

## Development

The project boots through a Vite ES-module entrypoint (`js/main.js`).

Core game/platform modules are loaded directly as ESM, including the main game loop module (`js/game.js`).

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Validation

```bash
npm run check
```
